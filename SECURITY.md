## Reporting a Vulnerability

Please do not report security vulnerabilities through public GitHub issues.

If you believe you have found a security vulnerability in DNS Manager, please report it via email to us. We will respond as soon as possible, usually within 48 hours.

Please include the following information in your report:
* The version of DNS Manager you are using
* Steps to reproduce the vulnerability
* What you expected to happen
* What actually happened

We take security issues very seriously and will work with you to understand and resolve the issue as quickly as possible.
